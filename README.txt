Este proyecto es parte de mis estudios acádemicos en la UNED de Costa Rica. 

Comprende el uso básico de las herramientas git, gitHub para el manejo de versiones y 
repositorios de un proyecto simple (Blog de PROIFED, una sección de la vicerrectoría
de investigación de la UNED de Costa Rica).

Este proyecto queda abierta al público, pero por utilizar una plantilla de pago
no se autoriza su reutilización como un blog propio.
